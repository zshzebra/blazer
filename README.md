## 🛠 Requirements

- Zig 0.13.0 or later
- Linux/macOS/Windows

> Tested on Linux only, other platforms are not guarenteed to work

## 🚧 Installation

### Clone the repository
```sh
git clone https://github.com/zshzebra/blazer.git
```

#### Build the project
```sh
cd blazer
zig build
```
